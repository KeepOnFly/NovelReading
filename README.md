建议环境：
intellij IDEA 2022.3.3
JDK 17

功能：
1. 设置页面
	1.1 选择配置文件保存目录，指定文件
	1.2 设置每页展示行数
	1.3 跳转指定页面
	1.4 设置阅读模式，翻页、滑动，设置滑动步长

2. 展示页面
	2.1 展示文本
	2.2 鼠标滚轮上下翻页
	2.3 上一页，下一页，隐藏
	
3. 特殊功能
	3.1 搜索
	3.2 识别章节名，跳转（未实现）
	3.3 保存阅读进度，默认跳转当前阅读进度
	
使用方式：
下一页（向下滑动）：N(按钮）、D(键盘）、→（方向键）、滚轮向后（鼠标）
上一页（向上滑动）：P(按钮）、A(键盘）、←（方向键）、滚轮向前（鼠标）
隐藏：H（按钮）、空格键（键盘）
全文阅读：F(键盘）
搜索：search(按钮，开始搜索）、next(按钮，下一个匹配项）
打开新书籍：open（按钮）
